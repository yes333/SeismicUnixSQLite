Overview of spFramework library

This library provides a basic C++ framework for programming data processing modules for the SU system by CWP (Center for Wave Phenomena of Colorado School of Mines).

This framework provides three groups of functionality:

  1. Basic exception and error message utilities, defined in SPBaseUtil.hh
  2. Segy trace data access utilities, defined in SPAccessors.hh
  3. The abstract processing class to be subclasses by each processing module class, defined in SPProcessor.cpp
